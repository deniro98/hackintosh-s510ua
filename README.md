    2.CPU:            Intel Core i3-7100U
    3.Graphic:        Intel HD Graphics 620 1536 МБ
    4.Wifi:           Intel Dual Band Wireless-AC 8265 - with bluetooth // REPLACED WITH DW1560 (AirDrop and Handoff Working perfectly)

	3. Reboot.
